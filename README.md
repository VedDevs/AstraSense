# AstraSense

A smart sensor module that: Measures temperature & humidity Measures air pressure Measures Air Quality

## Features

- Triple DHT11 sensing
- BMP180 pressure & altitude monitoring
- GP2Y air quality sensing
- OLED live Stats
- Serial Monitor output

## Hardware

- Arduino UNO/Nano
- 3× DHT11
- BMP180
- GP2Y Dust Sensor
- SSD1306 OLED Display

## Libraries Used

- DHT Library
- Adafruit BMP085
- Adafruit GFX
- Adafruit SSD1306
